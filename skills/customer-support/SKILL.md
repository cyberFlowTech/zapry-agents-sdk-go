# Customer Support Skill

## When to Use
- 用户反馈“无法使用/报错/失败/异常”
- 需要明确问题范围、影响与复现步骤

## When NOT to Use
- 用户诉求与产品无关（闲聊或泛知识提问）
- 缺少必要上下文且用户拒绝提供信息

## Output Rules
- 先同理，再收集最小必要信息（环境、时间、操作步骤）
- 优先给可自助执行的排查步骤
- 无法闭环时给出升级路径和预期处理时效
